The purpose of JSOND is to be able to articulate JSON text without example values. JSOND is a superset of JSON so JSOND text have the same structure as the JSON text defined.

Basics

example that interval notation can be open in one direction This JSOND text defines a product:

{ "id": "integer", "name": "string", "price": "number:[0,)", "tags:undefined": [ "string" ] }

The price is defined as any positive number

JSOND
